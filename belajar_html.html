Belajar HTML

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat dan merancang halaman web. HTML berfungsi untuk menyusun konten di dalam halaman web dengan menggunakan berbagai elemen atau tag, seperti teks, gambar, tautan, dan tabel, sehingga browser dapat menampilkan informasi secara terstruktur dan mudah diakses oleh pengguna.
